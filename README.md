# NanJingIPTV
本项目里的方案是：电脑利用双网卡直接播放江苏南京电信IPTV电视直播的组播源，以及回放节目

为了家里多台设备观看IPTV节目，比较流行方案是增加路由器或者交换机设备，部署 [rtp2httpd](https://github.com/stackia/rtp2httpd) 或者udpxy等软件，通过这些软件代理，将IPTV的组播源改成家庭局域网内的单播源。这个方案虽然成熟，但是要改变家庭组网。而我的个人诉求是只要能够在电脑上播放IPTV和回放就可以，不需要所有设备都能放，也不想增加网络设备。对我而言最简单的方案是电脑增加一个USB网卡，专门接入IPTV网络，实现电脑看电视的目的。

#背景：
建议先找AI问问IPTV组播是什么概念，这里不解释了。组播源的地址类似于这样  (CCTV1HD) rtp://239.49.8.19:9614

在电信的光猫上，互联网业务和IPTV业务是分离的，电脑接入互联网业务无法直接访问239开头的组播源。为解决两个网络隔离的问题，尝试过为电脑增加无线网卡，光猫为IPTV网络激活一个无线网，但是实测结果是隔墙衰减过于严重，约等于不可用。 所以我最终增加了一个有线的USB网卡，接入到光猫上绑定IPTV的网口。


#配置过程

1、用管理员账号登录光猫，网络设置里面2_Other_B_VID_43 就是IPTV业务，默认绑定了网口2，这里增加绑定网口4

2、电脑两个网卡，主板网卡接入Internet的网口，USB网卡接入光猫的网口4。

3、配置网络，让组播流量从USB走

   删除已有路由  
   >route delete 239.0.0.0

   查找USB网卡的索引号：
   >netsh interface ipv4 show interfaces

   假设查询到对应索引号为9， 为组播源增加一个永久路由
   >route add 239.0.0.0 mask 255.0.0.0 0.0.0.0 if 9 -p

   如果只是看电视直播，至此已经完成所有网络配置

4、查找可用的南京电信组播源

5、播放直播

使用Potplayer 打开m3u文件就可以选台观看直播了，但是不支持节目单。

Windows Store里面查找 progTV 下载安装3.11版本，可以根据m3u文件进行频道分组，支持节目单。 但是不支持回放点播。
